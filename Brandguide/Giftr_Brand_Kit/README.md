# Giftr Brand Kit

This folder contains a clean vector reconstruction of the selected **Giftr** logo system, plus PNG exports, app icons, favicons, brand guidelines, UI kit reference boards, copy snippets, and implementation tokens.

## Folder structure

- `01-logos/svg` — primary, stacked, icon, wordmark, compact, and reversed SVG logos.
- `01-logos/png` — PNG exports for quick use in decks, docs, and mockups.
- `02-monochrome` — black and white logo/icon variants.
- `03-app-icons` — square app icon exports at 1024, 512, 256, 128, and 64 px.
- `04-favicons` — favicon SVG, PNG sizes, and `.ico` file.
- `05-brand-guidelines` — brand guideline markdown, palette assets, and reference boards.
- `06-tokens` — color tokens as JSON, CSS variables, and Tailwind config snippet.

## Core positioning

**Giftr** is an autonomous AI gifting agent that searches, buys, pays for, and delivers digital gifts automatically.

Primary tagline:

> AI agents that buy and deliver gifts automatically.

## Primary logo usage

Use `logo-primary.svg` for most applications. Use the reversed logo on dark backgrounds and the icon-only mark for favicons, social avatars, and compact UI.

Maintain clear space around the mark. Do not stretch, recolor, add shadows, or remove the orange sparkle.

## Note

These assets are prepared for hackathon/demo use and product exploration. Before public launch, do a final brand/legal review and have a designer refine the master vector if needed.
